mac 使用记录

开始用mac，对于用了N年windows系统的我来说，是不小的问题，一点自己在用的时候整理的东东

全屏的快捷键 control+command+f

退出(完全退出)command+Q

关分页Command+W

刷新Command+R

新开分页Command+T

截屏键shift+Command+3全屏和+4局部

桌面快捷方式 finder =>command + option + 拖拽到桌面上

command +f3 回到桌面切换(非全屏模式下)

command+control+f 程序全屏切换

command+tab 程序切换

cmd+c→cmd+v 复制粘贴

cmd+c→cmd+option+v 剪切粘贴

command+q 完全退出程序
