hexo的用法介绍

安装：(我是用homebrew安装的)

node.js
git (全局安装)
npm install  -g hero-cli

应用 项目中

hero init
npm install
hexo new “name” 新建
hero generate 生成静态页面
hexo server 启动服务器

连接github

nam install heao-deployer-git —save

_config.yml
deploy:
type: git
repo: https://github.com/name/name.io.git
branch: master
这个东西格式很重要。

hero deploy 同步

常用命令

hero clean
hero generate （g）
hero deploy (d)

删除文章 在source->_posts 中把文章给删除了 再clean 然后重新hexo g ,hexo d就可以。
